# Solving Teukolsky Equation with Physics-Informed Neural Networks

## Project Structure

```
SolvingTeukolsky/
├── config/              # Configuration files
├── data/               # Raw data storage
├── dataset/            # Dataset processing and loading
├── model/              # Neural network model definitions
├── physical_ansatz/    # Physical ansatz and constraints
├── trainer/            # Training loops and procedures
├── utils/              # Utility functions
├── test/               # Unit tests
├── saved_models/       # Trained model checkpoints
├── outputs/            # Training outputs and results
├── .gitignore         # Git ignore rules
└── README.md          # This file
```

## Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
